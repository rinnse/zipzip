# zipzip
(5명의 팀원들과 만든 팀프로젝트)

압축 파일(zip)을 모티프로 하여 집(ZIP).zip 이라는 뜻을 담고 있음
이름과 같이 서울 소재의 다양한 부동산 정보를 담고 있으며
누구나 쉽고 간편하게 검색, 필터링, 업로드할 수 있는 사이트
